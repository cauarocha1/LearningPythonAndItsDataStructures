# Learning Python and Data Structures

Repositório de estudos em Python com exemplos práticos organizados por assunto, incluindo fundamentos da linguagem, estruturas de dados e funções.

## Objetivo

Consolidar a base de Python por meio de arquivos curtos e diretos, cada um focado em um conceito específico e realizar o curso "Aprendendo Python e suas estruturas de Dados - DIO"

## Estrutura do projeto

### `Fundamentals - 00`

Exemplos de base da linguagem:

- `hello_world.py`, `print_input.py`
- Tipos e conversões: `data_types.py`, `converting_types.py`
- Operadores: aritméticos, comparação, atribuição, lógicos, identidade e associação
- Estruturas condicionais: `conditional_structures.py`, `aligned_conditional_structures.py`, `ternary_conditional_structures.py`
- Laços de repetição: `repetition_structure_for.py`, `repetition_structure_while.py`, `repetition_structure_brake.py`
- Strings: `string_1.py` até `string_4.py` e desafios

### `DataStructure - 01`

Conteúdo de estruturas de dados e funções:

- `Lists`: declaração, acesso, slicing, compreensão, métodos (`append`, `extend`, `sort`, etc.)
- `Tuples`: acesso, slicing, iteração e métodos (`count`, `index`)
- `Sets`: operações (`union`, `intersection`, `difference`, etc.) e métodos principais
- `Glossary` (dicionários): criação, acesso, dicionários aninhados, iteração e métodos (`get`, `items`, `update`, etc.)
- `Functions`: retorno, argumentos nomeados, `*args`, `**kwargs`, escopo e parâmetros especiais
- `CollectionsAndFunctions`: desafios práticos integrando coleções e funções


## Perfil de uso

- Estudo incremental por arquivos pequenos.
- Consulta rápida de sintaxe e métodos de coleções.
- Prática de desafios com foco em lógica.
